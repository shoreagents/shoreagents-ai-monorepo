# üìã HOW TO GET YOUR SUPABASE CREDENTIALS:
# ==================================

## STEP 1: Get Database Connection Strings
1. Go to: https://supabase.com/dashboard
2. Select your project
3. Click: Settings (left sidebar) ‚Üí Database
4. Scroll to "Connection string" section
5. You'll see 4 modes - we need 2:

   A) For DATABASE_URL:
      - Click "Transaction" mode
      - Copy the entire string
      - It looks like: postgresql://postgres.xxxxx:[YOUR-PASSWORD]@aws-0-us-east-1.pooler.supabase.com:6543/postgres?pgbouncer=true
      - Add &connection_limit=1 at the end
      - Paste into DATABASE_URL above

   B) For DIRECT_URL:
      - Click "Session" mode
      - Copy the entire string  
      - It looks like: postgresql://postgres.xxxxx:[YOUR-PASSWORD]@aws-0-us-east-1.pooler.supabase.com:5432/postgres
      - Paste into DIRECT_URL above

## STEP 2: Get API Keys (Optional for now)
1. Click: Settings ‚Üí API
2. Copy "Project URL" ‚Üí paste into NEXT_PUBLIC_SUPABASE_URL
3. Copy "anon public" key ‚Üí paste into NEXT_PUBLIC_SUPABASE_ANON_KEY
4. Copy "service_role" key ‚Üí paste into SUPABASE_SERVICE_ROLE_KEY
