### ‚ùå "MCP Server not found"
**Solution**: Make sure you fully restarted Cursor (Quit & Reopen)

### ‚ùå "Cannot connect to n8n"
**Solution**:

2. Verify https://stepten.app.n8n.cloud/ is accessible

### ‚ùå "AI doesn't see workflows"
